# PBSTableView

QTableView with many extension

> 1. Filter Row
> 2. Footer Row
> 3. Column Auto Width
> 4. Enable/Disable Sort
> 5. Auto Edit
> 6. Quick Customize
> 7. Add New Row
> 8. Delete Row
> 9. Column Alignment
> 10. Column Formatting
> 11. Default Editor
> 12. Export Table (xlx, xml, Excel Xmli Word Xml, CSV, PDF,Text)
> 13. Print Table
> 14. Load/Save State
> 15. Filter Box
> 16. ReadOnly Columns
> 17. Focused Columns
> 18. Move Column

And Many Extension
